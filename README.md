## 🛠️ Installation Steps

1. Clone the repository

```bash
git clone https://github.com/Achiaga/star-wars-heroes.git
```

2. Change the working directory

```bash
cd star-wars-heroes/
```

3. Install dependencies

```bash
yarn install
```

4. Run the app

```bash
yarn start
```

🌟 You are all set!

## 🧪 Run Tests

yarn test
